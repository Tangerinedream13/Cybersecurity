# Cybersecurity — CSCI 364

This repository contains programming assignments for my Cybersecurity course, covering classical cryptography, memory exploitation, and steganography — implemented in C.

## Programming Assignments

| Lab | Assignment | Topics |
|-----|-----------|--------|
| [lab1](./lab1) | PRG1 — Caesar Cipher | ROT-13, substitution ciphers, modular arithmetic |
| [lab2](./lab2) | PRG2 — Buffer Overflows | Stack memory, unsafe input, exploit basics |
| [lab3](./lab3) | PRG3 — Vigenère Cipher | Polyalphabetic substitution, key-based encryption |
| [lab4](./lab4) | PRG4 — Beale Cipher Intro | Book ciphers, index-based encoding |
| [lab5](./lab5) | PRG5 — Beale Cipher | Full Beale implementation with real-world texts |
| [lab6](./lab6) | PRG6 — Steganography Intro | LSB image encoding, hiding data in JPEGs |
| [lab7](./lab7) | PRG7 — Steganography Images | Multi-image steganography, PNG secret extraction |

## Topics Covered

- Classical cryptography (Caesar, Vigenère, Beale ciphers)
- Buffer overflow vulnerabilities and memory exploitation
- Steganography — concealing data within image files
- C programming: pointers, file I/O, bitwise operations
